# Istiak Abedin's Portfolio Website

A modern, responsive portfolio website built with HTML5, CSS3, and vanilla JavaScript. This portfolio showcases your ML/NLP expertise and web development projects.

## 📁 Project Structure

```
portfolio/
├── index.html           # Main HTML file
├── style.css           # Styling and responsive design
├── script.js           # Interactive functionality
├── README.md           # This file
└── pic/                # Profile photo folder
    └── profile.jpg     # Your profile picture (add this)
```

## 🚀 Quick Start

1. **Add Your Profile Photo**
   - Get a JPG or PNG photo of yourself
   - Copy it to the `pic/` folder
   - Rename it to `profile.jpg` (or `profile.png`)
   - Refresh your browser (F5) to see changes

2. **Open in Browser**
   - Double-click `index.html` to open in your default browser
   - Or right-click → Open with → Your preferred browser

## 🐛 Troubleshooting

### Profile photo not showing
- Check the photo is in `pic/` folder
- Verify filename is `profile.jpg` (or `.png`)
- Refresh browser (Ctrl+F5 for hard refresh)

### Styling looks off
- Clear browser cache (Ctrl+Shift+Delete)
- Make sure `style.css` is in the same folder as `index.html`

### Animations not working
- Ensure `script.js` is loaded (check browser console)
- Check Java Script is enabled in your browser
